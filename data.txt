| Status | Business meaning               | Exact Auto Reg scenario                                                                                               | Code reason                                                                                                                                                                                                           |
| ------ | ------------------------------ | --------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **AC** | **Application Complete**       | Auto Reg creates the AR application successfully and no SSA MSP denial/pre-screen condition is hit.                   | In `updateArApplicationForAidTable(...)`, if `mSPAppStatus` is **not** `"DP"`, code sets `ApplicationStatusCd = "AC"`.                                                                                                |
| **DP** | **Pending Application Denied** | Used in **SSA MSP** flow when pre-screen/business rules fail.                                                         | `mSPAppStatus` is used as the driver; when it is `"DP"`, `updateArApplicationForAidTable(...)` sets `ApplicationStatusCd = "DP"`. The same DP flag is also used later to generate the SSA MSP denial correspondence.  |
| **DN** | **Case Denied**                | Used in **MPE denial** path, not general GCI flow. When MPE denial switch becomes `Y`, app is moved to denied status. | Auto Reg has a dedicated denial update method `updateStatusArAppforAidTable(...)`; this is invoked from the MPE denial flow when `MpeDenialSw = 'Y'`.                                                                 |
| **CA** | **Complete Associated**        | When application is associated to a case as a **program add**.                                                        | In `insertArApplicationForAid(...)`, if `associationType == PROGRAM_ADD`, code sets `ApplicationStatusCd = "CA"`.                                                                                                     |
| **DE** | **Denied**                     | In this codebase, used in the **association** path when the association is **not** program-add.                       | In `insertArApplicationForAid(...)`, if association type is not `PROGRAM_ADD`, code sets `ApplicationStatusCd = "DE"`.                                                                                                |
