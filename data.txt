Use this Copilot prompt:

```text
As part of our DB2 migration initiative, analyze the entire codebase I provide.

Goal:
Generate a comprehensive report of every SELECT query that does NOT contain a ORDER BY clause.
Important:
Do not miss any class, file, method, mapper, repository, DAO, service, XML, properties file, stored procedure call wrapper, or dynamically constructed SQL.

Scope:
1. Search all source files recursively.
2. Identify every SELECT query, including:
   - inline SQL strings
   - multiline SQL strings
   - string-concatenated SQL
   - StringBuilder/StringBuffer SQL
   - prepared statements
   - named queries
   - MyBatis/iBatis XML queries
   - JPA/native queries
   - stored procedure related SELECTs if present
   - SQL in config/properties/YAML files
3. Exclude queries only if they clearly already contain `ORDER BY`.
4. For each missing `ORDER BY`, provide:
   - file path
   - class name
   - method name if available
   - line number range
   - full SELECT query
   - query type: static / dynamic / XML / annotation / config
   - tables involved
   - risk level for DB2 migration
   - suggested corrected query with appropriate `ORDER BY`
   - notes or assumptions

Output format:
Create a detailed report grouped by module/package/class.

Also provide:
1. Total number of SELECT queries found.
2. Total number missing ORDER BY.
3. List of files/classes with no SELECT queries.
4. List of files that could not be confidently analyzed.
5. Any dynamic SQL fragments that may form SELECT queries but need manual review.

Be exhaustive. Prefer false positives over missing queries. Do not summarize only; include the actual SQL queries.
```
